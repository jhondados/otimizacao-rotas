# 🚚 Otimização de Rotas (VRP)

> Otimização de rotas de entrega com OR-Tools e ML. Resolve VRP com restrições, reduz custo de combustível e melhora prazos de entrega.

**🌐 Demo ao vivo (web app interativo):** https://jhondados.github.io/otimizacao-rotas/

## 📊 Resultados-chave (KPIs)
**-24%** Custo · **28K** Rotas/dia · **-18%** CO2 · **+12%** Prazo

## 🧠 Tecnologias de IA
OR-Tools · Optimization · ML

## ✨ O que faz
- VRP com janelas de tempo
- Otimização de combustível
- Redução de CO2
- Reotimização dinâmica

## 🗄️ Fonte de dados real
[ANTT + OSM](https://www.gov.br/antt)

## ⚙️ Como funciona
1. **Coleta de dados reais** — integração com ANTT + OSM
2. **Processamento com IA** — OR-Tools · Optimization · ML
3. **Dashboard & insights** — KPIs, gráficos e recomendações acionáveis
4. **Ação automatizada** — alertas, predições e decisões em tempo real
